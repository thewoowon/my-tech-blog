# Linear Algebra

## part 1.2 - Vector Space

앞서 우리는 벡터에 대해서 공부했다.
그리고 평면에서의 8가지 성질에 대해서도 알아보았다.

<div className="px-4 py-2 my-4 bg-blue-100 border-l-4 border-blue-500">
  <div className="text-lg font-bold">Properties</div>
  <div>1. 모든 벡터 x, y에 대하여 x + y = y + x이다.</div>
  <div>2. 모든 벡터 x, y, z에 대하여 (x + y) + z = x + (y + z)이다.</div>
  <div>3. 모든 벡터 x에 대하여 x + 0 = x를 만족하는 벡터 0이 존재한다.</div>
  <div>4. 각 벡터 x마다 x + y = 0을 만족하는 벡터 y가 존재한다.</div>
  <div>5. 모든 벡터 x에 대하여 1x = x이다.</div>
  <div>6. 모든 실수 a, b와 모든 벡터 x에 대하여 (ab)x = a(bx)이다.</div>
  <div>7. 모든 실수 a와 모든 벡터 x, y에 대하여 a(x + y) = ax + by이다.</div>
  <div>8. 모든 실수 a, b와 모든 벡터 x에 대하여 (a + b)x = ax + bx이다.</div>
</div>

이제 이러한 성질을 가지는 벡터들의 집합을 정의해보자.

> 체 F에서의 벡터공간(Vector Space) 또는 선형공간(Linear Space) V는
> 다음 8가지 조건을 만족하는 두 연산, 합과 스칼라곱에 대한 닫힌(closed) 집합이다.

  <div>경계를 모두 포함하는 집합</div>
</div>
